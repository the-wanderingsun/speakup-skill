# SpeakUp — 职场英语口语 AI 教练

> 把这壶水，真正烧开。

一个帮助职场人士提升英语口语的 Claude Skill。不打基础、不纠音、直接开口练！

每次练完都能感受到一点点进步。

---

## 这个 Skill 是为谁做的

- 有英文面试、汇报、或和外国同事沟通需求的职场人
- 英语基础还在，但开口就卡、词汇量跟不上表达的人
- 试过很多 App 和 AI 工具，但坚持不下去的人

不适合完全零基础的用户。

---

## 它能做什么

**口语练习**
根据你的目标推荐职场场景话题，给出关键词提示，你说完之后 AI 给出贴合你水平的具体反馈。每次只改一两个点，不会让你觉得哪哪都错。

**个人词库 Flashcard**
每次练习结束后，自动把你卡壳的词、说错的词、可以升级的词记录下来。下次练习时追踪你有没有自然用出来。

**AI 改进追踪**
AI 记得你上次学了什么，下次练习里主动检测你有没有真正用上，用上了就告诉你。

**Deadline 陪练**
告诉 AI 你什么时候有面试或汇报，它帮你规划每天练什么，倒计时陪你准备。

**英语人格档案**
每完成 5 次练习，AI 生成一段关于你表达风格的叙述——用朋友的口吻描述你是一个什么样的表达者，记录你的成长轨迹。

## 安装方法

**方法一：Claude.ai 网页版**

1. 下载本仓库的 `speakup.skill` 文件
2. 打开 [claude.ai](https://claude.ai)，进入设置
3. 找到 Skills 选项，点击安装，上传文件
4. 安装完成后，在对话里说"我想练英语"即可触发

**方法二：Claude Code**

```bash
/skills install speakup.skill
```

**方法三：直接粘贴 Prompt（无需安装）**

如果你想在任何 AI 工具里使用，包括 Claude 语音模式，复制 `prompt.md` 里的内容粘贴到对话开头即可。

## 使用示例

```
你：我想练英语

SpeakUp：好！在我们开始之前，想先了解一下你。
         你练英语主要是为了什么场景？
         比如面试、和外国同事开会、还是其他？

你：我下个月有个外企面试

SpeakUp：明白了，距离面试还有约 30 天，我们重点练面试场景。
         今天先来一个自我介绍，3 分钟，我给你几个关键词，
         你直接用英语说就好。

         关键词：background / key experience / achievement / why this role

         准备好了就开始，说完告诉我 done。
```

## 背后的故事

这个 Skill 来自一次真实的挫败。

我在职场做了快两年，一直觉得英语不重要但不着急，团队里有人帮我补短板，日子过得挺好。

直到有一次内推面试了一个非常心仪的岗位，面试全程顺利，最后面试官说了一句"我们用英文聊一下吧"，我的大脑直接宕机了。

事后和很多朋友聊，发现这种感觉不只是我有。英语口语几乎是每个职场人心里共同的一根刺。

我不是不知道英语重要。但"知道"和"当真"是两件事！

所以我做了这个工具，先给自己用，也希望能帮到有同样感受的人。

如果你也在把这壶水烧开的路上，欢迎一起。
